# Room Name: Windows Command Line


 Windows vs Linux Command Comparison

###  System Information
- `ver` → Shows Windows version  
- `systeminfo` → Detailed system information  
- `set` → Displays environment variables  

Linux equivalents:
- `uname -a`, `hostnamectl`, `env`

---

### 🌐 Network Commands
- `ipconfig` → View IP configuration  
- `ipconfig /all` → Detailed network info  
- `ping` → Test connectivity  
- `tracert` → Trace network route  
- `nslookup` → DNS queries  
- `netstat` → View network connections  

---

### 📁 File & Directory Management
- `dir` → List files  
- `cd` → Change directory  
- `mkdir` → Create directory  
- `rmdir` → Remove directory  
- `tree` → Show directory structure  
- `type` → Display file content  
- `more` → Paginated output  

Linux equivalents:
- `ls`, `pwd`, `cat`, `less`

---

### 📄 File Operations
- `copy` → Copy files  
- `move` → Move or rename files  
- `del` / `erase` → Delete files  
- `*` → Wildcard usage  

---

### ⚙️ Process Management
- `tasklist` → View running processes  
- `tasklist /FI` → Filter processes  
- `taskkill /PID` → Kill a process  

Linux equivalents:
- `ps`, `top`, `kill`

---

### 🔌 System Control
- `shutdown /s` → Shutdown system  
- `shutdown /r` → Restart system  
- `shutdown /a` → Abort shutdown
